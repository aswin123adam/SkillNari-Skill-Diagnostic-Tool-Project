## üöÄ Technical Architecture

### **Backend (The Intelligence)**

Built with **Python & Flask**, the backend functions as a high-performance API:

* **Skill Extraction:** Uses Natural Language Processing (NLP) including NER (Named Entity Recognition) and GloVe embeddings to understand the context of user skills.
* **Matching Algorithm:** Performs a high-speed comparison between user profiles and thousands of job requirements to calculate a "Match Score."
* **Recommendation Engine:** Queries a curated database of courses (Coursera, Udemy, edX) to find the exact match for identified skill gaps.

### **Frontend (The Experience)**

Built with **React.js**, focusing on a "Sustainable & Eco-friendly" UI/UX:

* **Dynamic Search:** Real-time job filtering based on user skill input.
* **Gap Visualization:** Clear, actionable tables showing missing skills and direct links to courses.
* **SEO Optimized:** Implemented `react-helmet-async` for metadata management.
